# Wallet Inspector 🔐

A blockchain wallet analysis web application.

## Features

- Ethereum wallet balance lookup
- Bitcoin wallet balance lookup
- Recent transactions
- Public wallet analysis
- Responsive dark interface

## Technologies

- HTML5
- CSS3
- JavaScript
- Etherscan API
- Blockstream API

## Supported Networks

Currently:

✅ Ethereum  
✅ Bitcoin  

Coming soon:

- Polygon
- BNB Chain
- Solana

## Security

This application only uses public blockchain addresses.

Never enter:

- Seed phrases
- Private keys
- Passwords
